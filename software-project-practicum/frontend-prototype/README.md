# 静态前端原型

这是 `software-project-practicum` 第一轮前端静态页面原型，当前已完成：

- `总览 /dashboard`
- `评分标准 /rubrics`
- `学生列表 /students`
- `学生详情 /students/anon-003`

## 当前目标

- 先确认教师视角的信息结构和视觉风格
- 先做静态页面，不接后端
- 先验证“总览 -> 列表 -> 详情”这条核心链路

## 启动方式

在该目录下执行：

```bash
npm install
npm run dev
```

## 当前说明

- 页面全部使用静态 mock 数据
- 顶部已展示批次级操作入口：上传作业、开始批改、导出 Excel、刷新结果
- 当前这些按钮仅做展示，不执行真实流程
- 评分标准页当前只展示“列表 + 编辑区预览”的静态结构
- 详情页以 `anon-003` 为主样例，突出“薪资结算进度跟踪未覆盖”等问题的展示方式

## 下一步建议

- 补充统计分析页
- 将 mock 数据进一步拆分
- 再接入 Spring Boot 提供的真实接口
